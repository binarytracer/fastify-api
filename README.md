## Fastify API

> practice project for fastify

### Inclusions

- [x] **Dotenv**, environment variables.
- [x] **Swagger**, api docs
- [x] **Typebox**, validation

### Local setup

1. pull the repo
2. run npm install
3. create a file `.env` from `env.copy`
4. run the application,

```sh
$ npm start
```
